vertex_texcoord

This function sets the texture coordinates to use for the vertex currently being defined for the custom primitive.

You supply the vertex buffer to write the data into as well as the UV position within the texture to use.

 

Syntax:

vertex_texcoord(buffer, u, v);

Argument Type Description
buffer Vertex Buffer The vertex buffer to write the information to.
u Real The first texture coordinate to use (0 - 1).
v Real The second texture coordinate to use (0 - 1).

 

Returns:

N/A

 

Example:

vertex_texcoord(buff, 0, 0);

The above code sets the UV values of the vertex being defined.
