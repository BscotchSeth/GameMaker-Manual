extension_get_version

This functions gets the version of an Extension Asset given its name and returns it as a string formatted as "GMEXT_<extensionName>_version".

 

Syntax:

extension_get_version(ext_name);

Argument Type Description
ext_name String The name of the extension

 

Returns:

String (formatted as "GMEXT_<extensionName>_version")

 

Example:

var _version_string = extension_get_version("MyExtension");
if (_version < 4)
{
    show_debug_message("This extension must be version >= 4 in order for the game to work correctly.");
}

The above code first calls extension_get_version to get a string containing the version number of an extension named "MyExtension". It stores the returned value in a temporary variable _version_string.
